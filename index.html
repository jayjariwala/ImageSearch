Image Search Microservice

  • Pass a query as a parameter and get the image URLs, alt text and thumbnail image for a set of images relating to a given search string.
  • Paginate through the responses by adding a ?offset=10 parameter to the URL.
  • Get a list of the most recently submitted search strings.
  • Visit that shortened URL, and it will redirect to the original link.

Example

  • /imageapi/grumpy%20cat?offset=2
  • /latest - get latest Searches
